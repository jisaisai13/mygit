# html-css 课堂笔记

## html文档

### 一个html5文件它的结构是什么

答：
- 文档声明`<!DOCTYPE html>`
- 根元素`html`
- 根元素中包含`head`和`body`
- `head`中包含`charset`编码、标题等等。
- `body`中是写代码的。

### 可以使用emmet插件快写html5文档

输入一个英文状态的`!`，然后按`tab`，即可快速创建`html5`文档。

### gbk编码和utf-8编码的区别

`gbk`国标码，仅包含文字、符号、大小写英文字母、数字。
`utf-8`万国码，包含`gbk`，还包含亚洲一些文字和特殊字符号。

当前最流行的是`utf-8`编码,设置是在`head`区域

```html
<head>
<meta charset="utf-8">
</head>
```
